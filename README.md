# TweeterMeter
## Are you a positive or negative tweeter? How emotional of a tweeter are you?

TweeterMeter is a web app that enables users to analyze the sentiment of any Twitter user's recent tweets.

Still in progress.

## Technologies

### Frontend
* JavaScript
* React
* Redux
* CSS
* HTML

### Backend
* Express
* Node.js
* Twitter API
* Google Natural Language Processing API
